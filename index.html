Teste

Sobre o CoronaTeste:

Se você deseja detectar, se você foi infectado pelo covid-19 ou não, você está no lugar certo. Basta responder às perguntas e descobrir o resultado. Lembre-se de que fiz isso para a minha pratica com programação e o resultado desse teste não deve ser levado a sério.

Responda a pergunta a seguir

1. Você esta sentindo febre(superior a 37,5 graus)?

Marque uma alternativa

2. Você tem dificuldade para respirar?

Marque uma alternativa

3. Você costuma se sentir cansado, mesmo que não faça nada?

Marque uma alternativa

4. Você está sentindo algumas dores musculares?

Marque uma alternativa

5. Você esta tossindo frequentemente?

Marque uma alternativa

6. Você tem dor de garganta?

Marque uma alternativa

Resultado do teste:

Se precisar tirar qualquer dúvida, ligue para o Disque Saúde, pelo número 136.

Serviços

Orientações:

Lave as mãos

A lavagem frequente das mãos é a principal recomendação para se prevenir.

Evite contatos físicos

Vamos nos previnir, evite contato físico, principalmente com pessoas do grupo de risco!

Use álcool em gel

Quando nao puder lavar as mãos com água e sabão, faça o uso do álcool em gel para higienizar as suas mãos.

Fique em casa

A forma mais eficaz de barrar o Covid-19 é ficando em isolamento social. só saia quando for necessário
